Идея

Algorithm NextBinaryVector(v)

Input: binary vector v of length n

1. m ← index of the rightmost 0 in v
2. if such position does not exist
3.     return LAST_VECTOR
4. v[m] ← 1
5. for j ← m+1 to n
6.     v[j] ← 0
7. return v
Генерация всех векторов
Algorithm GenerateAllBinaryVectors(n)

1. v ← (0,0,...,0)
2. add v to result

3. while true
4.     v ← NextBinaryVector(v)
5.     if v is LAST_VECTOR
6.         break
7.     add v to result

8. return result
