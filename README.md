## Генерация всех подмножеств конечного множества (Вариант 1)

###Задача размещения баз.

Некоторая территория разделена на
n районов. Предполагается, что военная база, расположенная в каком-
то районе, может контролировать не только этот район, но и соседние,
граничащие с ним районы. Каждый район характеризуется определен-
ными убытками, связанными с расположением в нем военной базы.
Требуется найти такие места размещения военных баз, чтобы суммар-
ные убытки были минимальные и вся территория была под контролем.


- $n$ — количество районов
- $c_i$ — убытки от размещения базы в районе $i$
- $A = \{a_{ij}\}$ — матрица смежности, где:
  $$a_{ij} = \begin{cases} 
  1, & \text{если район } i \text{ граничит с районом } j \\
  0, & \text{в противном случае}
  \end{cases}$$
- $x_i$ — двоичная переменная, где:
  $$x_i = \begin{cases} 
  1, & \text{если в районе } i \text{ размещена база} \\
  0, & \text{в противном случае}
  \end{cases}$$

### Целевая функция:
Минимизировать суммарные убытки:

$$\min \sum_{i=1}^{n} c_i x_i$$


$$
\begin{aligned}
& \text{минимизировать} && \sum_{i=1}^{n} c_i x_i \\
& \text{при ограничениях} && \sum_{j=1}^{n} a_{ij} x_j \geq 1, \quad i = 1,\ldots,n \\
& && x_i \in \{0, 1\}, \quad i = 1,\ldots,n
\end{aligned}
$$


1. Начинаем с вектора $(0, 0, \ldots, 0)$
2. Последовательно прибавляем двоичную единицу
3. Заканчиваем вектором $(1, 1, \ldots, 1)$

Число всех возможных вариантов:

$$\sum_{k=0}^{n} \binom{n}{k} = 2^n$$

где $\binom{n}{k} = \frac{n!}{k!(n-k)!}$ — число сочетаний из $n$ по $k$.

$$x = (1, 0, 0, 0, 0, 1, 1, 0, 0)$$

Минимальное число баз: **3**
